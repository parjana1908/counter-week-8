# counter-week-8
A simple and interactive React Counter Application built using functional components and React Hooks. This app allows users to increment, decrement, and reset a counter with basic validation to prevent negative values.
 # Features
Increment the counter value
Decrement with minimum limit (0) validation
Reset counter to default (0)
Displays alert message when limit is reached

# Tech Stack
React (Hooks - useState)
JavaScript (ES6)
HTML & CSS

# Usage
Click Increment to increase count
Click Decrement to decrease count (won’t go below 0)
Click Reset to set counter back to 0

# Purpose

This project demonstrates basic React concepts like:

State management using useState
Event handling
Conditional rendering
